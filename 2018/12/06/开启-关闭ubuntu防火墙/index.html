开启/关闭ubuntu防火墙

发表于 2018-12-06 | 分类于 Linux

安装防火墙

使用方法

1、启用

sudo ufw enable
sudo ufw default deny
作用：开启了防火墙并随系统启动同时关闭所有外部对本机的访问（本机访问外部正常）。

2、关闭

sudo ufw disable

3、查看防火墙状态

sudo ufw status

4、开启/禁用相应端口或服务举例

推荐设置
sudo apt-get install ufw
sudo ufw enable
sudo ufw default deny

这样设置已经很安全，如果有特殊需要，可以使用sudo ufw allow开启相应服务。
